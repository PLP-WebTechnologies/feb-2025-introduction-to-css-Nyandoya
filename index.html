Cascading Style Sheets

CSS (Cascading Style Sheets) controls how HTML elements appear on a webpage.

By using CSS, you can control the color, font, size, spacing, and many other aspects of HTML elements.

CSS can be added to HTML documents in three ways:

  1. Inline
  2. Internal
  3. External
